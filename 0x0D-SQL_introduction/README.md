		**SQL - Introduction**

SQL stands for Structured Query Language.

SQL can perform various operations on data, such as creating, updating, deleting, querying, joining, aggregating, etc.
SQL is also a declarative language, which means that it focuses on what data to retrieve or modify, rather than how to do it.
